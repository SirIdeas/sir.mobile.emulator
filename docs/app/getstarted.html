Comenzar

Cómo descargar, configurar y comenzar a usar Sir Mobile Emulator en tu aplicación HTML5.

Descargar

Existe dos formas de usar Sir Mobile Emulator. Descargando los archivos comprimidos o incluyendo los mismos desde un CDN.

Descargar comprimido

Puede descargar los archivos JS, CSS e imágenes utilizadas en un compromido listo para ser usado.

Descargar

Incluir desde CDN

De igual forma para muestras en la WEB se puede incluir desde sitios CDN.



              
Obtener de GitHub

              

                Si desea indagar profundamente en el código de Sir Mobile Emulator puede clonar los fuentes desde el GitHub.
