Sir Mobile Emulator

Emulador visual de dispositivos móviles actuales para probar y presentar aplicaciones HTML5.

Descargar
Versión actual: v1.0.2
Sir Mobile Emulator es una extensión de jQuery que permite observar como luce tu aplicación en las pantallas de los dispositivos de mayor uso en la actualidad, con la finalidad de hacer de esta responsiva. Además podrás realizar presentaciones de tu aplicación antes de compilar nativamente para cada SO, o liberar versiones de pruebas online para navegadores webs, en los diferentes dispositivos móviles de mayor uso en la actualidad.
