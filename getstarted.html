Comenzar

Cómo descargar, configurar y comenzar a usar Sir Mobile Emulator en tu aplicación HTML5.

Descargar

Existen dos formas de usar Sir Mobile Emulator. Descargando los archivos comprimidos o incluyendo los mismos desde la web.

Descargar comprimido

Puede descargar los archivos JS, CSS e imágenes utilizadas en un comprimido listo para ser usado.

Descargar

Incluir desde la web

De igual forma para muestras en la en línea se puede incluir desde sitios web.

 
Obtener de GitHub
 
 Si desea indagar profundamente en el código de Sir Mobile Emulator puede clonar los fuentes desde el GitHub.
